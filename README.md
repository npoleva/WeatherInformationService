# Сервис аналитики погоды WeatherInformationService

**Cервис для сбора, кэширования и визуализации данных о погоде**

## Основные функции

- **Получение погоды** по названию города
- **Кэширование в Redis** на 15 минут (чтобы не дергать API при каждом запросе)
- **Построение графика** температуры по часам
- **Автоматическое определение координат** города

## Технологии

- Java + Spring Boot
- Redis (для кэширования)
- OpenMeteo API 
- Библиотека для построения графиков

## Как работает

1. Приходит запрос на `/weather?city=London` (или любой другой город)
2. Проверяем кэш в Redis
3. Если данных нет или устарели:
   - Получаем координаты города (широта, долгота)
   - Запрашиваем погоду по координатам
   - Сохраняем в Redis на 15 минут
4. Возвращаем данные погоды + график температуры
